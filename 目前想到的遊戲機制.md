# 目前想到的遊戲機制

## 時間

- [x] 玩家做的行動會增減時間

   - 例如：走到公園花了三分鐘、睡午覺花了三十分鐘

- [ ] 有些事件需要在一定的時間內才會觸發

- [ ] 有些地點在過了一定的時間會出現不同內容（例如：這裡打烊了。）

- [x] 隨著時間流逝體力也會做增減。

## 隨機事件

- [ ] 玩家滿足特定條件可能會觸發某些事件

## NPC故事

- [ ] NPC的主線

- [ ] 查看NPC個人資料頁面（需要對話才能知道某些資訊）

- [ ] 與NPC的好感度

## 玩家

- [ ] 玩家在這個世界觀的聲望（可能會影響人際關係、知名度之類的）

- [ ] 玩家的統計頁面（統計玩家遊戲內容，例如：經歷過Ｘ次搶匪、約會過幾次…）

- [ ] 玩家的技能

## 未來功能(優先度還不高)

- [ ] NPC有生日？想在遊戲內放日曆，玩家也比較好觀察他未來有什麼事情要做

- [ ] 紙娃娃系統

- [ ] 開始頁面 (遊戲設定調整等等)

- [ ] 遊戲時間換日自動存檔
